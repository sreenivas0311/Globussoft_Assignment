# Globussoft Assignment

This repository contains solutions for the Globussoft  assignment.

## Tasks Included

### Task 1 - Amazon Laptop Scraper

A Python-based web scraping project that extracts laptop information from Amazon and stores the results in a CSV file.

Folder:

```text
Task1_Amazon_Scraping/
```

Technologies:

* Python
* Requests
* BeautifulSoup
* Pandas

---

### Task 2 - Face Authentication

A face verification system built using FastAPI and InsightFace.

The application accepts two face images, extracts embeddings using a pretrained model, calculates cosine similarity, and determines whether the images belong to the same person.

Folder:

```text
Task2_Face_Authentication/
```

Technologies:

* Python
* InsightFace
* ONNX Runtime
* OpenCV
* NumPy
* FastAPI
* Uvicorn
